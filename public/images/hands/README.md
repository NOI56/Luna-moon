# 📁 รูปมือสำหรับเกมเป่ายิงฉุบ

## 📋 ไฟล์ที่ต้องมี

ใส่รูปมือในโฟลเดอร์นี้:

1. **rock.png** - มือกำ (Rock)
   - มือกำเป็นกำปั้น
   - ขนาดแนะนำ: 200x200px หรือใหญ่กว่า
   - พื้นหลังโปร่งใส (PNG)

2. **paper.png** - มือเปิด (Paper)
   - มือเปิดทั้งหมด
   - ขนาดแนะนำ: 200x200px หรือใหญ่กว่า
   - พื้นหลังโปร่งใส (PNG)

3. **scissors.png** - มือชู 2 นิ้ว (Scissors)
   - มือชูนิ้วชี้และนิ้วกลาง
   - ขนาดแนะนำ: 200x200px หรือใหญ่กว่า
   - พื้นหลังโปร่งใส (PNG)

4. **default.png** - มือปกติ (Default)
   - มือปกติสำหรับสถานะเริ่มต้น
   - ขนาดแนะนำ: 200x200px หรือใหญ่กว่า
   - พื้นหลังโปร่งใส (PNG)

## 🎨 ข้อกำหนด

- **รูปแบบไฟล์:** PNG (รองรับพื้นหลังโปร่งใส)
- **ขนาดแนะนำ:** 200x200px ขึ้นไป
- **พื้นหลัง:** โปร่งใส (transparent)
- **สไตล์:** ตามที่ต้องการ (realistic, cartoon, etc.)

## 📝 หมายเหตุ

- ถ้าไม่มีไฟล์ → ระบบจะใช้ emoji fallback
- รูปจะถูก resize อัตโนมัติให้พอดีกับขนาดที่กำหนด
